### 给初学者
如果你刚开始接触OpenGL，建议从glut学起。GLFW虽然功能强大，性能领先，但网上教程极少且散。GLFW的使用还需要配合GLSL语法，不是朝夕即可学会的。因此建议从glut用起。

如果有恒心，请和我继续下去。
